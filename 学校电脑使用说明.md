# 班级积分管理系统 - 学校电脑使用说明

## 🚨 常见问题：点击没反应

### 原因分析

学校电脑通常有以下限制导致程序无法运行：

1. **杀毒软件拦截**（最常见）
2. **缺少系统运行库**
3. **系统权限限制**
4. **GPU/显卡兼容性问题**

---

## ✅ 解决方法

### 方法一：以管理员身份运行

1. 右键点击程序图标
2. 选择 **"以管理员身份运行"**
3. 如果弹出用户账户控制(UAC)提示，点击"是"

### 方法二：添加到杀毒软件白名单

#### 360安全卫士
1. 打开 360 → 防护中心 → 信任与阻止
2. 点击"添加信任文件"
3. 选择程序exe文件

#### 火绒安全
1. 打开火绒 → 安全设置 → 病毒防护 → 信任区
2. 点击"添加文件夹"
3. 选择程序所在文件夹

#### Windows Defender
1. 设置 → 更新和安全 → Windows安全中心
2. 病毒和威胁防护 → 管理设置
3. 排除项 → 添加排除项 → 文件夹
4. 选择程序所在文件夹

### 方法三：安装运行库

如果提示缺少 dll 文件，需要安装：

**Visual C++ Redistributable**
- 下载地址：https://aka.ms/vs/17/release/vc_redist.x64.exe
- 下载后双击安装

**.NET Framework 4.8**
- Windows 7/8 可能需要安装
- 下载地址：https://dotnet.microsoft.com/download/dotnet-framework/net48

### 方法四：使用学校专用版

我们提供了专门针对学校电脑优化的版本：

**文件：`班级积分管理系统宠物版-学校版.exe`**

特点：
- 禁用GPU加速（解决显卡兼容问题）
- 禁用沙盒模式（解决权限限制）
- 禁用悬浮球功能（避免被拦截）
- 详细的错误日志记录

---

## 🔍 如何查看错误信息

程序会在同目录下生成 `app-debug.log` 日志文件，记录启动过程。

如果程序无法启动，请查看此文件中的错误信息。

---

## 📋 学校电脑检查清单

在运行程序前，请确认：

- [ ] 已尝试右键"以管理员身份运行"
- [ ] 已将程序添加到杀毒软件白名单
- [ ] 程序路径不含中文或特殊字符（建议放在 D:\app\ 目录）
- [ ] 已安装 Visual C++ 运行库
- [ ] 使用的是学校专用版本

---

## 🆘 仍无法解决？

请提供以下信息：

1. **操作系统版本**（Windows 7/8/10/11）
2. **错误截图**（如有弹窗）
3. **app-debug.log 文件内容**
4. **是否安装杀毒软件**（360/火绒/其他）

联系技术支持时提供以上信息，可以更快解决问题。
