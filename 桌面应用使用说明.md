# 班级积分管理系统宠物版 - 桌面应用使用说明

## 两种桌面应用方案

本项目支持两种桌面应用打包方案：**Electron** 和 **NW.js**

---

## 方案一：Electron（推荐）

### 特点
- ✅ 功能最强大，生态最完善
- ✅ 支持 Windows、Mac、Linux
- ✅ 可以打包成安装程序（.exe/.dmg/.AppImage）
- ✅ 自动更新支持
- ⚠️ 打包体积较大（约 100-200MB）

### 安装依赖

```bash
# 安装 Electron 和打包工具
npm install
```

### 运行方式

```bash
# 开发模式运行
npm run start

# 或
npx electron .
```

### 打包构建

```bash
# 打包所有平台
npm run build

# 仅打包 Windows
npm run build:win

# 仅打包 Mac
npm run build:mac

# 仅打包 Linux
npm run build:linux

# 打包成便携版（不解压安装）
npm run pack
```

### 输出文件

打包后的文件位于 `dist-electron` 目录：
- Windows: `班级积分管理系统宠物版 Setup 3.8.0.exe`
- Mac: `班级积分管理系统宠物版-3.8.0.dmg`
- Linux: `班级积分管理系统宠物版-3.8.0.AppImage`

---

## 方案二：NW.js

### 特点
- ✅ 架构简单，配置方便
- ✅ 打包体积相对较小
- ✅ 启动速度快
- ✅ 支持 Windows、Mac、Linux
- ⚠️ 功能相对 Electron 简单
- ⚠️ 生态不如 Electron 丰富

### 安装依赖

```bash
# 安装 NW.js
npm install nw --save-dev
npm install nw-builder --save-dev
```

### 运行方式

```bash
# 方式1：使用 package.json 中的配置
npm run nw

# 方式2：直接使用 nw 命令
npx nw .

# 方式3：使用 nw-main.js 入口（带菜单）
npx nw . --main nw-main.js
```

### 打包构建

```bash
# 打包所有平台
npm run nw:build

# 或
npx nwbuild -p win64,osx64,linux64 .

# 仅打包 Windows
npm run build:win

# 仅打包 Mac
npm run build:mac

# 仅打包 Linux
npm run build:linux
```

### 输出文件

打包后的文件位于 `dist-nwjs` 目录：
- Windows: `classroom-pet-manager-win-x64/`
- Mac: `classroom-pet-manager-osx-x64/`
- Linux: `classroom-pet-manager-linux-x64/`

## 快速开始

### 第一步：安装 Node.js

确保已安装 Node.js（建议版本 18+）：
```bash
node --version
npm --version
```

### 第二步：安装依赖

在项目根目录执行：
```bash
npm install
```

### 第三步：运行应用

**Electron 方案：**
```bash
npm run start
```

**NW.js 方案：**
```bash
npm run nw
```

### 第四步：打包（可选）

根据需要选择打包命令（见上文）。

## 常见问题

### Q: 打包后的应用无法显示宠物图片？
A: 确保 `pet` 文件夹已正确包含在打包文件中。Electron 和 NW.js 的配置已包含此文件夹。

### Q: 如何修改窗口大小？
A: 
- **Electron**: 修改 `electron-main.js` 中的 `width` 和 `height`
- **NW.js**: 修改 `package.nw.json` 中的 `window.width` 和 `window.height`

### Q: 如何更改应用图标？
A: 替换 `pet/仓鼠/1.png` 为您自己的图标文件，并更新配置文件中的路径。

### Q: 数据存储在哪里？
A: 
- **Electron**: 使用 Chromium 的本地存储（IndexedDB/localStorage）
- **NW.js**: 同样使用 Chromium 的本地存储
数据默认保存在用户的应用数据目录中。

### Q: 如何启用开发者工具？
A:
- **Electron**: 取消注释 `electron-main.js` 中的 `mainWindow.webContents.openDevTools()`
- **NW.js**: 按 F12 或点击菜单"视图" → "开发者工具"
